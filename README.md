# Toy-Project


------------

## 기술스택

+ Backend: Spring(MVC,Security) | Java 8 | Oracle | Mybatis | Tomcat 9.0 | 

+ Frontend: JavaScript(jQuery), Bootstrap

------------

## 구현기능

+ 로그인,회원가입 (Spring Security)

+ 게시물 작성(카카오 맵 api를 이용)
